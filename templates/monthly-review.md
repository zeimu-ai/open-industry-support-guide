# 月次面談準備テンプレート

> このプロンプトテンプレートは、freee MCP + Claude/ChatGPT で月次面談の準備を行うためのものです。
> `{業種}` `{顧問先名}` を置き換えて使用してください。

## プロンプト

```
あなたは税理士事務所のAIアシスタントです。
以下の顧問先の月次面談準備を行ってください。

## 顧問先情報
- 顧問先名: {顧問先名}
- 業種: {業種}
- 決算月: {決算月}

## 手順

### Step 1: 試算表データの取得
freee MCPで当月と前月の試算表（BS/PL）を取得してください。

### Step 2: 業種別着眼点との照合
以下のファイルを参照してください:
- 人間向け: industries/{業種ファイル名}.md（例: industries/01-food-service.md）
- AI向けJSON: data/{業種ファイル名}.json（例: data/01-food-service.json）
  ※ JSONの sections.financials に経営指標、sections.visit に訪問時チェックポイントがあります

以下の指標を確認してください（原典由来の観点）:

- 原価率（業界目安との比較）
- FL比率（飲食業の場合）
- 人件費率
- 前月比の変動が大きい科目

### Step 3: 面談アジェンダの作成
以下の構成で面談アジェンダを作成してください:

1. **今月のハイライト**: 特筆すべき変動（3項目以内）
2. **業種別チェック**: 業界指標との比較結果
3. **確認事項**: 経営者に聞くべき質問（2-3問）
4. **改善提案**: 数値に基づく具体的な提案（あれば）

出力は日本語で、税理士が経営者に説明しやすい平易な言葉で記述してください。
```

## 使い方

### Claude + freee MCP の場合

1. Claude Codeまたはclaude.aiでfreee MCPを接続
2. 上記プロンプトの `{業種}` `{顧問先名}` `{決算月}` を置き換え
3. `industries/` の該当業種mdファイルも一緒に読み込ませる
4. freee MCPで試算表を取得 → 業種別着眼点と照合 → アジェンダ生成

### ChatGPT の場合

1. ChatGPTに `industries/` の該当業種mdファイルをアップロード
2. 試算表のCSV/PDFもアップロード
3. 上記プロンプトを入力

## 業種別の重点チェック項目

| 業種 | 重点指標 | 目安 |
|------|---------|------|
| 飲食業 | FL比率 | 60%以下 |
| 飲食業 | 原価率 | 30%前後 |
| 小売業 | 粗利率 | 業態による |
| 建設業 | 完成工事総利益率 | 15-25% |
